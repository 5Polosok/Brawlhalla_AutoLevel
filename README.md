# AutoEXP Bot (неофициально)

Автокликер /бот для фарма,  
написан для **личного использования**

Работает через:
- поиск картинок (OpenCV + pyautogui)
- эмуляцию клавиатуры и мыши
- немного логики и много `sleep()`

---

## Что тут есть

**AutoEXP.py** — основной бот  
**gui_tool.py** — простая GUI-шка на Tkinter  
**images/** — все картинки, по которым бот ориентируется  
**levels/** — сюда сохраняются скриншоты уровней   

---

## Возможности

- автоматическое создание лобби
- настройка лобби
- добавление бота и хендикапы
- старт/выход из матча
- ожидание конца боя
- скриншоты уровня
- отправка в Discord через webhook
- GUI с кнопками Start / Stop
- отдельная кнопка для скрина персонажа (`char_unhovered.pnh`)
---
